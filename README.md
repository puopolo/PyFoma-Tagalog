# Morphological Analyzer with FST for Tagalog
This task was completed for Morphology NLP, instructed by Dr. Mans Hulden (EHU/UPV HAPLAP 2023-2024).

## Task
The task was to construct a morphological analyzer for a language using finite-state techniques.
The purpose was to explore some basic phenomena of word formation in some language and implement them
as a finite-state transducer.
I chose to make a small toy grammar of __Tagalog__ noun and verb inflection.
While the grammar is small, much larger grammars can be built using the same design principles. 

 
 ## Tools
 - Regular expressions
 - __PyFoma__, PyFoma, an open-source Python module for constructing weighted and unweighted finite-state transducers and automata from regular expressions, string rewriting rules, right-linear grammars, or low-level state/transition manipulation.
